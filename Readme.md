 ``Usage: chaos_game.exe [OPTIONS]``

 ```
 Options:
  -x <X>                         [default: 1000]
  -y <Y>                         [default: 1000]
  -e, --equidistant <PTS>        [default: 3]
  -p, --points <COORDINATES>...
  -n, --numerator <NUM>          [default: 1]
  -d, --denominator <DENO>       [default: 2]
  -i, --iter <ITERS>             [default: 10000]
  -h, --help                     Print help
  ```
